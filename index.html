Pi Cycle

The Pi Cycle graph uses a 111-day moving average (111DMA) and 350-day moving average multiplied by 2 (350DMAx2). In this graph, the top green area is above the indicator value of 0.95, i.e., when the value of the 111DMA graph crosses almost above twice the value of the 350DMA. The lower red area marks an indicator value of less than 0.35.
